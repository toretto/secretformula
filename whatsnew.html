Updates

Change log of the latest updates and changes

2019 07 02

We had to make changes RE: our CSS, because it would break things in an upcoming version that'll be coming to smaller screens near you. We are making note of the fact that the app doesn't look fantastic on tablets

Today, we made the first build of the Android and Windows app. Spoiler: That's when the CSS decided not to play nice. Expect us to announce an Android version soon(ish)

2019 05 12

We made changes in how input is processed, so that both the correct and the wrong way of writing decimals are accepted by the app. The right way, of course, is using the comma.

We made a style change to the front-end of the app and made the button colors more consistant. Aren't you glad it only took us a year to to that?

Developers' note:I am trying to figure out how this app works so I can expand on it. Not working on it for over a year might not have been the brightest idea.

2018 04 26

A new formula has been added, which allows you to split the costs of items when some participanes only "used" the rental for a part of the time. E.G Someone joins you in your awesome cottage half-way the vacation, and you want to calculate the share he needs to pay.

This was our most complex formula by far, for various reasons. It's also the hardest to explain the use off, but we hope that some of you find a use for it.

2018 04 18

Translation of the app into English is now complete. While translating we found a duplicate formula, which we converted to a new formula: You can now calculate the price without tax starting from a price that includes taxes or VAT. I mean, if you are into that sort of thing.
We have also started working on a way to allow you to use both comma's and dots as decimale separators. As a result, the formula "Buy X, Get Y Free" will not be working under Microsoft Edge (it'll always return "0" as a result). This won't be fixed. Get your shit together, Edge.

2018 04 16

We started translating the app to the Queens' language after giving up on the idea of building a language dictionary into the app. For now.

2018 04 06

We made design changes so that the app is more "usable" on big screens (aka desktop devices) and immediately had to implement a fix because our buttons would become hard to read. We also ported the changes from the app to the web version so that absolute URL's are no longer being used.

2018 04 04

Well, it's been a while but we're back with some 'exciting' changes. (Let's be honest. You just want this thing to work)

  • We added a new formula, that helps you calcute fuel costs for a trip.
  • We made changes to how the app laods CSS (Yet to be applied to all pages) so you don't need to load stuff from other parties. This makes things faster!
  • We changed a class name to make future development simpler.
  • We started working on an Apple / Android version!
  • We've found ways to cut the time neede to add an formula in half.
